# project-planner

simple project planner in vue.
json-server to mock api and data

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and hot-reloads for development

```
json-server --watch data/db.json
```

### Compiles and minifies for production

```
npm run build
```
